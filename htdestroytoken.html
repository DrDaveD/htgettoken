NAME

htdestroytoken − remove bearer and vault tokens

SYNOPSIS

htdestroytoken [-q]

DESCRIPTION

htdestroytoken removes a bearer token found by WLCG Bearer Token Discovery and also removes a vault token found in the default location used by htgettoken.

Note that the vault server additionally caches bearer tokens, so the next time that htgettoken is run the same bearer token might be returned unless a new one is forced to be retrieved with either oidc authentication or with an htgettoken −−minsecs option.

OPTIONS

The following options are recognized:

−q

Do removals silently
